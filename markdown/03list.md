# 列表

*Markdown支持有序列表和无序列表*

### 无序列表

*无序列表使用(\*) 、+ 、- 作为标记*

* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项



###  有序列表

_有序列表使用数字并加上. 来表示，如：_

1. 第一项
2. 第二项
3. 第三项



### 列表嵌套

_列表嵌套只需在子列表中的选项添加四个空格即可:_

1. 第一项：

        - 第一项嵌套的第一个元素
        - 第二项嵌套的第二个元素

2. 第二项：

      - 第二项嵌套的第一个元素

      - 第二项嵌套的第二个元素
